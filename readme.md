# This project involves a comprehensive analysis of the Titanic dataset
This repository contains my analysis and predictive modeling for the famous Titanic dataset. The goal of this project is to predict which passengers survived the Titanic shipwreck using various features like age, gender, and passenger class.

## 📂 Project Structure

| Folder/File | Description |
| :--- | :--- |
| data/ | Contains the Titanic dataset (titanic.csv). |
| reports/ | PDF reports and visualizations of the findings. |
| titanic_dataset.ipynb | The main notebook containing the full analysis. |

## 📊 Objective
The Titanic was a "unsinkable" ship that sank in 1912, leading to the loss of 1502 out of 2224 passengers and crew. While there was some element of luck involved in surviving, it seems some groups of people were more likely to survive than others.

In this project, I explore:
* Data Cleaning: Handling missing values for 'Age' and 'Cabin'.
* EDA: Visualizing survival rates based on Class, Sex, and Age.
* Feature Engineering: Creating new features like 'Family Size'.
* Modeling: Training a classifier to predict survival.

## 🛠️ Tools Used
* Python (Pandas, NumPy)
* Matplotlib & Seaborn (for data visualization)
* Scikit-Learn (for Machine Learning)

## 📈 Results
Current model accuracy: 82.9%
